# My Mobile App

## Project Overview
Event manager application to view, register, and manage events.

## Features
- View events
- Search for events
- Register to events
- Create, update, delete events

## Installation
1. Clone the repo
2. cd to omarpro
3. Run `npm install`
4. Run `npx expo start`
5. Press 'w' to run in web view
